# <a name="c6505"></a>C6505

 Diese Warnung gibt an, dass die MustCheck-Eigenschaft f端r einen void-Datentyp verwendet wurde. Die MustCheck-Eigenschaft kann nicht f端r den void-Typ verwendet werden. Entfernen Sie entweder die MustCheck-Eigenschaft, oder verwenden Sie einen anderen Datentyp.


# <a name="c6383"></a>C6383
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C6383: Puffer체berlauf aufgrund der Konvertierung einer Element Anzahl in eine Byte Anzahl: f체r den Parameter \<number> im Aufrufen von wird eine Element Anzahl erwartet. \<function>  
  
 Diese Warnung gibt an, dass eine nicht konstante Byte Anzahl 체bermittelt wird, wenn eine Element Anzahl erforderlich ist. Dies tritt normalerweise auf, wenn eine Variable mit `sizeof` einem Typ multipliziert wird, aber die Code Analyse schl채gt vor, dass eine Element Anzahl erforderlich ist.  
  
## <a name="example"></a>Beispiel  
 Der folgende Code generiert diese Warnung:  
  
```  
#include <string.h>  
  
void f( wchar_t* t, wchar_t* s, int n )  
{  
  // code...  
  wcsncpy (t, s, n*sizeof(wchar_t)); // warning 6383  
  // code ...  
}  
```  
  
 Um diese Warnung zu beheben, Multiplizieren Sie die Variable nicht mit `sizeof` einem Typ, wie im folgenden Code gezeigt:  
  
```  
void f( wchar_t* t, wchar_t* s, int n )  
{  
  // code   
  wcsncpy (t, s, n);  
  // code ...  
}  
```  
  
 Der folgende Code korrigiert diese Warnung mithilfe der sicheren Zeichen folgen-Manipulations Funktion:  
  
```  
void f(wchar_t* t, wchar_t* s, size_t n)  
{  
  // code...  
  wcsncpy_s( t, sizeof(s), s, n );  
  // code...  
}   
```

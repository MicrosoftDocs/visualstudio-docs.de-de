# <a name="c28109"></a>C28109
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C28109: Variable kann nicht zum Zeitpunkt des Funktionsaufrufs enthalten sein.  
  
 Das Programm enth√§lt eine Ressource, die beim Aufrufen dieser Funktion nicht gespeichert werden sollte. In der Regel ist dies ein Hinweis darauf, dass die Ressource versehentlich zweimal abgerufen wurde. Das Code Analysetool meldet diese Warnung, wenn Ressourcen in derselben Funktion abgerufen und freigegeben werden.  
  
## <a name="example"></a>Beispiel  
 Im folgenden Codebeispiel wird diese Warnung generiert:  
  
```  
ExAcquireResourceLite(resource, true);  
...  
ExAcquireResourceLite(resource, true);  
```  
  
 Im folgenden Codebeispiel wird diese Warnung vermieden:  
  
```  
ExAcquireResourceLite(resource, true);  
```

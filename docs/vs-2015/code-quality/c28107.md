# <a name="c28107"></a>C28107
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C28107: Die Ressource muss beim Aufrufen der Funktion enthalten werden.  
  
 Eine Ressource, die das Programm vor dem Aufrufen der Funktion abrufen muss, wurde nicht abgerufen, als die Funktion aufgerufen wurde. Folglich schl√§gt der Funktionsaufrufe fehl. Diese Warnung wird nur gemeldet, wenn Ressourcen in derselben Funktion abgerufen und freigegeben werden.  
  
## <a name="example"></a>Beispiel  
 Im folgenden Codebeispiel wird diese Warnung generiert:  
  
```  
ExAcquireResourceLite(resource, true);  
ExReleaseResourceLite(resource);  
```  
  
 Im folgenden Codebeispiel wird diese Warnung vermieden:  
  
```  
KeEnterCriticalRegion();  
ExAcquireResourceLite(resource, true);  
ExReleaseResourceLite(resource);  
KeLeaveCriticalRegion();  
KeEnterCriticalRegion();  
ExAcquireResourceLite(resource, true);  
ExReleaseResourceLite(resource);  
KeLeaveCriticalRegion();  
```

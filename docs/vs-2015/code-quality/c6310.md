# <a name="c6310"></a>C6310
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

 Diese Meldung gibt an, dass eine unzul채ssige Konstante im Filter Ausdruck eines strukturierten Ausnahme Handlers erkannt wurde. Die Konstanten, die f체r die Verwendung im Filter Ausdruck eines strukturierten Ausnahme Handlers definiert sind, lauten wie folgt:

## <a name="example"></a>Beispiel  
 Der folgende Code generiert diese Warnung:  
  

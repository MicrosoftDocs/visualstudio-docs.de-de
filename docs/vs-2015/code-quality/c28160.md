# <a name="c28160"></a>C28160
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C28160: Fehleranmerkung  
  
 Diese Warnung wird gemeldet, wenn eine Anmerkung `__drv_error` gefunden wurde. Diese Anmerkung wird zum Kennzeichnen von Codierungs Methoden verwendet, die korrigiert werden sollten, und kann mit einer Anmerkung verwendet werden, `__drv_when` um bestimmte Kombinationen von Parametern anzugeben.

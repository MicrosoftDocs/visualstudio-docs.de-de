# <a name="c28284"></a>C28284
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C28284: Für das Symbol werden Prädikate für Nicht-Funktionssymbole momentan nicht unterstützt.  
  
 Diese Warnung weist darauf hin, dass eine bedingte Anmerkung (Prädikat `__drv_when` ) für einen anderen als eine Funktion gefunden wurde.

# <a name="c28194"></a>C28194
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C28194: Die Funktion wurde als Aliasbenennung des Werts in der Variablen deklariert und wurde ohne die Aliasbenennung beendet.  
  
 Diese Warnung gibt an, dass der Funktionsprototyp für die zu analysierende Funktion über eine Anmerkung verfügt `__drv_isAliased` , die angibt, dass Sie das angegebene Argument als *Alias* angibt (d. h., der Wert wird so zugewiesen, dass er die Rückgabe von der Funktion überdauern wird). Die Funktion gibt jedoch kein Alias für das Argument an dem Pfad an, der durch die Anmerkung angegeben wird. Die meisten Funktionen, die eine Variable Aliasnamen speichern, speichern ihren Wert in einer globalen Datenstruktur.

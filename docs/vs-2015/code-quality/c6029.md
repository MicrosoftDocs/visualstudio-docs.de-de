# <a name="c6029"></a>C6029

[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C6029: möglicher Pufferüberlauf im-Befehl \<function> : Verwendung von deaktiviertem Wert  
  
Diese Warnung zeigt an, dass einer Funktion, die einen Puffer und eine Größe annimmt, eine nicht überprüfte Größe übergeben wird. Die aus einer externen Quelle eingelesenen Daten wurden nicht daraufhin überprüft, ob sie kleiner als die Puffergröße sind. Ein Angreifer kann versuchen, absichtlich einen erheblich größeren als den erwarteten Wert für die Größe anzugeben, was zu einem Pufferüberlauf führt.  
  
Überprüfen Sie grundsätzlich beim Lesen von Daten aus einer nicht als vertrauenswürdig eingestuften externen Quelle die Gültigkeit der Daten. Es ist normalerweise sinnvoll, die Größe zu prüfen, um sicherzustellen, dass sie im erwarteten Bereich liegt.  
  
## <a name="example"></a>Beispiel  

Der folgende Code generiert diese Warnung, indem die [Infodatei](/windows/win32/api/fileapi/nf-fileapi-readfile) mit Anmerkungen in der Funktion zweimal aufgerufen wird. Nach dem ersten Aufruf markiert die Attributeigenschaft „Post“ den zweiten Parameterwert als nicht vertrauenswürdig. Daher löst die Übergabe eines nicht vertrauenswürdigen Werts im zweiten Aufruf von `ReadFile` diese Warnung aus, wie im folgenden Code dargestellt:  
  
```cpp

#include "windows.h" 

bool f(HANDLE hFile)  
{  
    char buff[MAX_PATH];  
  
    DWORD cbLen;  
    DWORD cbRead;  
  
    // Read the number of byte to read (cbLen).  
    if (!ReadFile (hFile, &cbLen, sizeof (cbLen), &cbRead, NULL))    
    {  
        return false;  
    }  
    // Read the bytes  
    if (!ReadFile (hFile, buff, cbLen, &cbRead, NULL))  // warning 6029  
    {  
        return false;  
    }  
  
    return true;  
}  
```  
  
Um diese Warnung zu korrigieren, überprüfen Sie die Puffergröße, wie im folgenden Code dargestellt:  
  
```cpp
bool f(HANDLE hFile)  
{  
    char buff[MAX_PATH];  
  
    DWORD cbLen;  
    DWORD cbRead;  
  
    // Read the number of byte to read (cbLen).  
    if (!ReadFile (hFile, &cbLen, sizeof (cbLen), &cbRead, NULL))    
    {  
        return false;  
    }  
    // Ensure that there's enough space in the buffer to read that many bytes.  
    if (cbLen > sizeof(buff))  
    {  
        return false;  
    }  
    // Read the bytes  
    if (!ReadFile (hFile, buff, cbLen, &cbRead, NULL))  // warning 6029  
    {  
        return false;  
    }  
  
    return true;  
}  
```

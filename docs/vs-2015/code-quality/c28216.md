# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28216: die \_ Check_return \_ Anmerkung gilt nur für nach Bedingungen für den bestimmten Funktionsparameter.  
  
 Die-Anmerkung wurde `_Check_return_` in einem anderen Kontext als angewendet `__post` . möglicherweise wird ein- `__post` `__drv_out` Modifizierer (oder) benötigt, oder er kann falsch platziert werden.

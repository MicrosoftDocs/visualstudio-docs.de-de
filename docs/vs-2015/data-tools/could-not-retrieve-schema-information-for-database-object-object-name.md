# <a name="could-not-retrieve-schema-information-for-database-object-ltobject-namegt"></a>Schema Informationen für Objektname des Datenbankobjekts konnten nicht abgerufen werden. &lt;&gt;
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Diese Meldung wird in der Regel angezeigt, wenn ein Objekt in **Server-Explorer/Datenbank-Explorer** in die Zwischenablage kopiert, aus der Datenbank gelöscht und dann auf dem Designer eingefügt wird. Da das Datenbankobjekt nicht mehr vorhanden ist, wird diese Meldung angezeigt.

### <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler

- Klicken Sie auf **OK**, um das Dialogfeld zu schließen.

# <a name="list-of-available-services"></a>Liste der verfügbaren Dienste
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

[!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] und das Visual Studio SDK unterstützen die folgenden Dienste. Einige Pakete bieten eigene Dienste, die hier nicht aufgelistet sind – beispielsweise verfügen Sprachdienste nicht über eine einzige Dienst-GUID. Sie müssen den Namen der Sprache verwenden, um die GUID des sprach Dienstanbieter in der Registrierung zu ermitteln.  
  
 Verwenden Sie die hier aufgelisteten oder von einer anderen Quelle (z. b. Sprachdienste) abzurufenden Dienst-GUIDs, um die primäre Schnittstelle zu erhalten, die mit den einzelnen Diensten angezeigt wird.

# <a name="hierarchies-and-selection"></a>Hierarchien und Auswahl

In diesem Abschnitt werden die Konzepte der [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] Hierarchien und der Auswahl erläutert.

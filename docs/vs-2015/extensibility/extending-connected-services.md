# <a name="extending-connected-services"></a>Erweitern von verbundenen Diensten
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Mit dem Visual Studio verbundene Dienste SDK können Dienstanbieter Visual Studio-Erweiterungen erstellen, die App-Entwickler problemlos zu Ihrem Projekt hinzufügen können, ohne die Visual Studio-IDE zu überlassen.

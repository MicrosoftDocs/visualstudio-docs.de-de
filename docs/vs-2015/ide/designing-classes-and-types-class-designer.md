> [!NOTE]
> Wenn Sie mit Klassen-Designer arbeiten, sollten Sie den ausgegebenen Code vor der Ausführung immer überprüfen, um sicherzustellen, dass der Code für Ihren Sicherheitskontext geeignet ist.

# <a name="flowchart-activity-designers"></a>Flussdiagramm-Aktivitätsdesigner
[!INCLUDE[wfd1](../includes/wfd1-md.md)] umfasst eine Reihe von vom System bereitgestellten Aktivitäten, die Sie beim Erstellen von Workflows verwenden können. Dieser Abschnitt enthält die vom System bereitgestellten Aktivitäten, die zum Verwalten von Flussdiagrammen innerhalb eines Workflows dienen. In den folgenden Themen werden diese Aktivitäten beschrieben und Anleitungen zu ihrer Verwendung bereitgestellt.

## <a name="in-this-section"></a>In diesem Abschnitt
 [Flussdiagramm](../workflow-designer/flowchart-activity-designer.md) **zum Erstellen von Workflows, die komplexe Fluss Steuerungen definieren und verwalten.**  [FlowDecision](../workflow-designer/flowdecision-activity-designer.md) Stellt eine Verzweigung für den Ablauf von Steuerelementen in einer von zwei Alternativen bereit, je nachdem, ob eine angegebene Bedingung erfüllt ist.

 [FlowSwitch \<T> ](../workflow-designer/flowswitch-t-activity-designer.md) Stellt Verzweigungen für die Ablauf Steuerung basierend auf dem Übereinstimmungs Kriterium bereit, wenn mehr als zwei alternative Verzweigungen erforderlich sind.

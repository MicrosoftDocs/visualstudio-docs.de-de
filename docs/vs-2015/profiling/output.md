# <a name="output"></a>Output

Die Option **Ausgabe** gibt den Namen der Profilerstellungsdaten f체r die Leistungssitzung an. **Ausgabe** muss zusammen mit der Option **Start** verwendet werden.  
  
## <a name="syntax"></a>Syntax  
  
```  
VSPerfCmd.exe /Start:Method /Output:FileName [Options]  
```  
  
#### <a name="parameters"></a>Parameter  
 `FileName`  
 Der Name der Datendatei. Es werden vollst채ndige und partielle Pfade angenommen. Wenn kein Pfad angegeben ist, wird die Datei im aktuellen Verzeichnis erstellt.  
  
## <a name="required-options"></a>Erforderliche Optionen  
 Die Option **Ausgabe** muss zusammen mit der Option **Start** verwendet werden.  
  
 **Start:**`Method`  
 Gibt den Ausgabedateinamen an.

```  
VSPerfCmd.exe /Start:Sample /Output:TestApp.exe.vsp  
```

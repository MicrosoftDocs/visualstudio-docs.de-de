# <a name="idiastackwalkframeget_registervalue"></a>IDiaStackWalkFrame::get_registerValue
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Ruft den Wert eines Register ab.  
  
## <a name="syntax"></a>Syntax  
  
```cpp#  
HRESULT get_registerValue (   
   DWORD      index,  
   ULONGLONG* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `index`  
 in Ein Wert aus der [CV_HREG_e Enumeration](../../debugger/debug-interface-access/cv-hreg-e.md) -Enumeration, die das Register angibt, für das der Wert festgelegt werden soll.  
  
 `pRetVal`  
 vorgenommen Gibt den aktuellen Wert des Register zurück.  
  
## <a name="return-value"></a>Rückgabewert  
 Wenn die Ausführung erfolgreich ist, wird `S_OK`, andernfalls ein Fehlercode zurückgegeben.

# <a name="ca0505"></a>CA0505

Die veraltete Eigenschaft wird ignoriert, da die superceding-Eigenschaft definiert ist.

Warnung CA505 wird generiert, wenn sowohl die veralteten als auch die ersetzte Code Analyse Eigenschaften Werte in einer Projekt-oder buildkonfigurationsdatei aufweisen. Um dieses Problem zu beheben, entfernen Sie die veraltete-Eigenschaft.

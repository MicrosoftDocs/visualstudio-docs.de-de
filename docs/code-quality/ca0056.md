# <a name="ca0056"></a>CA0056
Eine Projektdatei weist eine falsche Version des Analysetools auf.

Diese Warnung tritt auf, wenn FxCopCmd.exe manuell mit einer Projektdatei mit einer falschen Code Analysetool-Version aufgerufen wird.

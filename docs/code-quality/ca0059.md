# <a name="ca0059"></a>CA0059
Fehler beim Wechseln der Befehlszeile.

Dieser Fehler tritt auf, wenn ein erforderlicher Befehls Zeilenschalter fehlt, z. b.

der Schalter/out oder/Console ist nicht angegeben. Dieser Fehler tritt auch auf, wenn ein ungültiger Schalter vorhanden ist, z. b. ein ungültiger Timeout Wert.

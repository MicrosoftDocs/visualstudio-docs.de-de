## <a name="im-unable-to-download-or-activate-visual-studio-code"></a>Ich kann Visual Studio Code nicht herunterladen oder aktivieren

Visual Studio Code ist ein kostenloser Code-Editor, für dessen Nutzung Sie kein kostenpflichtiges Visual Studio-Abonnement benötigen. Sie können Visual Studio Code direkt auf der Visual Studio-Website [herunterladen](https://code.visualstudio.com/download).

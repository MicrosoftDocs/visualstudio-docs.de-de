## <a name="im-unable-to-download-or-activate-visual-studio-community"></a>Ich kann Visual Studio Community nicht herunterladen oder aktivieren
Visual Studio Community ist die kostenlose, einfache Version der IDE, für deren Nutzung Sie kein kostenpflichtiges Visual Studio-Abonnement benötigen. Sie können Visual Studio Community direkt auf der Visual Studio-Website [herunterladen](https://visualstudio.microsoft.com/vs/community/).
